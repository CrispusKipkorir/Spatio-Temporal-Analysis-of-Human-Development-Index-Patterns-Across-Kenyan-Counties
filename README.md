# Spatio-Temporal Analysis of HDI Patterns Across Kenyan Counties

An interactive web-based platform for exploring human development and poverty indicators across Kenya's 47 counties.

## Features
- Choropleth mapping of 6 indicators (HDI, Poverty Headcount, MPI, etc.)
- Temporal slider (2015–2030) with national values for past/future years
- County selection and side-by-side comparison
- Export map as PNG, JPG, or PDF
- Responsive design with collapsible panels
- Detailed indicator explanations and data sources

## Data Sources
- County-level data (~2025): KNBS, OPHI, UNDP
- National trends: UNDP Human Development Reports, KNBS Economic Surveys

## Technology
- Leaflet.js – Interactive maps
- html2canvas & jsPDF – Map export
- Pure HTML/CSS/JavaScript – No backend required
